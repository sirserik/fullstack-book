## Ключевая концепция: Мультиязычность

В проекте MyApp используется уникальная система переводов — все тексты хранятся в таблице `translates`:

```
┌─────────────────────────────────────────────────────────────────────────┐
│                         СИСТЕМА ПЕРЕВОДОВ                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   Таблица: translates                                                   │
│   ┌────────┬─────────────────────┬──────────────┬──────────────────┐    │
│   │ id     │ ru                  │ en           │ kz               │    │
│   ├────────┼─────────────────────┼──────────────┼──────────────────┤    │
│   │ 1      │ "Добро пожаловать"  │ "Welcome"    │ "Қош келдіңіз"   │    │
│   │ 2      │ "Новости"           │ "News"       │ "Жаңалықтар"     │    │
│   │ 3      │ "О нас"             │ "About us"   │ "Біз туралы"     │    │
│   └────────┴─────────────────────┴──────────────┴──────────────────┘    │
│                                                                         │
│   Таблица: news                                                         │
│   ┌────────┬──────────────┬──────────────────────┬────────────┐         │
│   │ id     │ name (FK)    │ background_image     │ date       │         │
│   ├────────┼──────────────┼──────────────────────┼────────────┤         │
│   │ 1      │ 1            │ /storage/img/1.jpg   │ 2024-01-15 │         │
│   │ 2      │ 2            │ /storage/img/2.jpg   │ 2024-01-20 │         │
│   └────────┴──────────────┴──────────────────────┴────────────┘         │
│                 │                                                       │
│                 └──────► Связь с translates через name (FK)             │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## API структура

Структура проекта предоставляет публичный REST API:

| Endpoint | Описание |
|----------|----------|
| `GET /api/mainPage` | Главная страница |
| `GET /api/news` | Список новостей |
| `GET /api/news?year=2024&month=1` | Фильтрация новостей |
| `GET /api/aboutUs` | О университете |
| `GET /api/bachelorSchool` | Факультеты |
| `GET /api/contacts` | Контакты |
| `GET /api/headerNavBar` | Меню шапки |
| `GET /api/footerNavBar` | Меню футера |
| `GET /api/metaData` | SEO данные |

## Требования к окружению

### Backend
- PHP 7.3+ или 8.0+
- Composer
- MySQL 5.7+
- Apache/Nginx

### Frontend
- Node.js 18+
- npm или yarn

### Для деплоя
- VPS сервер (Ubuntu 20.04+)
- Nginx
- PM2 (для Node.js)
- Supervisor (для Laravel очередей)
- Let's Encrypt (SSL)
