## Быстрый старт

### Backend
```bash
cd app-backend
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
# API доступно на http://localhost:8000/api
```

### Frontend
```bash
cd app-frontend
npm install
npm run dev
# Сайт доступен на http://localhost:3000
```
